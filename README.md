# CheckMyStrength

**Discover Strength Through Action.**

CheckMyStrength is an evidence-based student guidance system that replaces passive reflection with active measurement. 

## Features
- **The Arena**: A set of micro-challenges to test real-world skills.
- **Friction Analysis**: Measures "Time vs. Accuracy" and "Friction" to determine cognitive suitability for tasks.
- **Guidance Engine**: Generates specific, data-backed reports on a student's dominant strengths and weaknesses.
- **Cyber-Minimalist UI**: A premium, dark-mode focused interface designed for focus.

## Technology
- Vanilla HTML5, CSS3, JavaScript (ES6+)
- LocalStorage for state persistence
- Responsive Glassmorphism Design

## How to Deploy
1. Upload these files to your GitHub repository.
2. Connect your repository to [Vercel](https://vercel.com).
3. Deploy!
